Welcome to the HTML Boilerplate Test

This is a simple test of the HTML boilerplate. It is a simple template that can be used to create a new HTML file. It includes the basic structure of an HTML file, including the doctype, html, head, and body tags. It also includes some basic meta tags, a link to a CSS file, and a header and nav section.
